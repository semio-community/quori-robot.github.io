Modules

Review each module and how it fits into Quori’s modular configurations.

Head

Core sensing and expressive interaction surface.

Pairs with mobile or stationary deployments
Mount point for speaker option

Speaker

Audio output module for speech and interaction cues.

Can mount on head, torso, base, or stand

Torso

Adds stability and mounting space for front/back modules.

Supports chest fascia and arm mounts
Typical bridge between base/stand and head

Chest

Front fascia for sensors, displays, and branding.

Sits on the front of the torso
Optional with arms

Arms

Optional manipulation modules mounted behind the torso.

Can be used with or without chest
Mounted slightly behind

Mobile Base

Locomotion platform for navigation and mobile studies.

Provides mobility + power
Supports torso/head stacks

Stand

Stationary base option for fixed-position experiments.

Swap-in for base configurations
Stable, compact footprint
